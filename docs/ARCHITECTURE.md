## Security Model

| Layer | Mechanism |
|-------|-----------|
| User вҶ’ CloudFront | HTTPS only (redirect HTTP) |
| User вҶ’ AgentCore | Cognito JWT (allowedClients validation) |
| AgentCore вҶ’ Neptune | VPC isolation + Security Group (port 8182, VPC CIDR only) |
| AgentCore вҶ’ S3 | IAM Role (GetObject, PutObject, DeleteObject, ListBucket) |
| AgentCore вҶ’ Bedrock | IAM Role (InvokeModel, InvokeModelWithResponseStream) |
| Neptune | Private subnet, no public endpoint, IAM auth disabled (POC) |
| S3 Buckets | BlockPublicAccess, OAI for CloudFront read-only |
| ECR | IAM-gated pull (Runtime Role), push (CodeBuild Role) |
